Tentang Saya

Ronald Parsaulian Simanjuntak

Mahasiswa Informatika | Universitas Pamulang

Saya adalah mahasiswa Universitas Pamulang yang memiliki ketertarikan besar di bidang Informatika, khususnya dalam keamanan siber (Cyber Security). Ketertarikan ini mendorong saya untuk terus mempelajari dan memahami perkembangan teknologi informasi, dengan fokus pada pengamanan sistem, jaringan, dan data.

Saat ini, saya sedang fokus mempelajari berbagai bahasa dan teknologi pemrograman yang relevan dengan cyber security, seperti Python untuk scripting dan automation, HTML/CSS/JavaScript untuk web security, serta tools seperti Kali Linux, Metasploit, Wireshark, dan Burp Suite. Saya juga mempelajari konsep-konsep penting seperti penetration testing, network security, cryptography, dan ethical hacking.

Melalui proses pembelajaran ini, saya berusaha meningkatkan kemampuan analisis keamanan, identifikasi vulnerability, dan implementasi defensive security measures. Saya memiliki keinginan kuat untuk berkontribusi dalam melindungi infrastruktur digital dan menjadi profesional cyber security yang kompeten. Dengan semangat belajar dan konsistensi, saya berharap dapat menerapkan ilmu yang saya pelajari ke dalam proyek nyata dan siap menghadapi tantangan di bidang keamanan teknologi informasi.
